GearBox Bot

Privacy Policy

Last updated: July 12, 2026

This Privacy Policy explains what information GearBox ("the Bot") collects, how it is used, and your rights regarding that data.

1. Information We Collect

GearBox collects only the minimum data required to operate its features:

  • User IDs and usernames — used transiently for moderation commands (/kick, /ban, /userinfo) and are not stored persistently.
  • Server (Guild) IDs — used to identify which server a command is being executed in.
  • Message content — read only when a message is edited, solely to log the before/after content to a designated channel configured by the server administrator.
  • Voice channel activity — when /play is used, the Bot joins a voice channel temporarily to stream audio. No audio from users is recorded.

2. How We Use the Information

  • To execute commands requested by server members or administrators.
  • To send moderation logs to a channel explicitly configured by the server administrator via /setlogchannel.
  • GearBox does not sell, share, or transmit any collected data to third parties.

3. Data Storage

GearBox does not maintain a persistent database of user data. The only data retained in memory is the logging channel ID per server, which is reset whenever the Bot restarts. No personal information is stored on disk.

4. Data Retention

Since no data is persisted, there is nothing to delete. Logs sent to a Discord channel are subject to Discord's own data retention policies.

5. Third-Party Services

  • Discord — the Bot operates within Discord's platform and is subject to Discord's Privacy Policy.
  • YouTube (via yt-dlp) — the /play command fetches publicly available audio streams. No user data is sent to YouTube.

6. Children's Privacy

GearBox is not directed at children under 13. We do not knowingly collect data from children. Discord's own Terms of Service require users to be at least 13 years old.

7. Changes to This Policy

We may update this Privacy Policy from time to time. Continued use of the Bot after changes constitutes acceptance of the revised policy.

8. Contact

If you have questions or concerns about this Privacy Policy, please reach out via Discord.
